```matlab
X = [ 1, 2, 3,
     -1, 0, 5,
     -2, 4, 9,]

result = X >= 0;

result = [ 1, 1, 1,
           0, 0, 1,
           0, 1, 1,]

-`X >= 0`会生成一个与X大小相同的逻辑矩阵  
-每个位置的值表示对应位置的元素是否大于等于0  
```

---

```
-线性代数中，向量的长度也叫模，范数。
```

```
denom(denom == 0) = 1

-`denom == 0`,生成逻辑数组。
-`denom(denom == 0)`意思是选择，denom中对应为true的所有元素。
-然后替换为1.
```
